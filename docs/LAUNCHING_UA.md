# Запуск для “будь-якого користувача” (Windows)

## Варіант 1 — ярлик на `run_arc.cmd` (найпростіше в розробці)

1. Встановіть залежності у `.venv` (як у README).
2. Створіть ярлик автоматично:
   - `powershell -ExecutionPolicy Bypass -File scripts\\windows\\create_shortcut.ps1`
3. Запускати з робочого столу: **ARC Platform**.

Скрипт `scripts\\windows\\run_arc.cmd` намагається використати `.venv\\Scripts\\pythonw.exe`, щоб не відкривалась консоль.

## Варіант 2 — один `.exe` (для розповсюдження без Python)

Рекомендовано: PyInstaller (або Nuitka). Потік такий:

1. Зібрати `arc` у `.exe`.
2. Упакувати у інсталятор (Inno Setup / NSIS) або у портативний `.zip`.
3. Інсталятор створює ярлики в Start Menu / Desktop.

## Варіант 3 — MSIX (сучасний “магазинний” підхід)

MSIX дає:
- автоматичні ярлики, деінсталяцію, пер-юзер інсталяцію
- більш “чистий” UX для не‑технічних користувачів

Потрібна підготовка пакету (manifest, підпис).
