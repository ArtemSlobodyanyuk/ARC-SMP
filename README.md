# ARC_Platform

Платформа розрахована на декілька видів редакторів для комфортної побудови карти
місцевості, міста, світу та ін.

### Головний хаб

Призначений для вибору платформи та різноманітних початкових функцій, є ввідним вікном для кожного користувача.

## ARC

Редактор для побудови схематичної карти. 
Розрахований на детальну схематичну розробки проекту
для інформативності, розуміння конструкції та контексту місцевості.

### Початкове меню

Меню містить в собі можливість почати новий проект, завантажити старий, вибір модифікацій.

### Редактор

Полотно для редагування карти

- Додавання об'єктів
- Зміна параметрів об'єктів
- Об'єднання об'єктів у групи
